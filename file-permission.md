#### command chmod 

chmod ย่อมาจาก "change mode"

#### ownership

- a = All
- u = User
- g = Group
- o = Other

#### file type

- <strong>-</strong> = File
- d = Directory
- l = Symbolic link

#### numberic code

- 0 = No Permission
- 1 = x = Execute หากเป็น file จะสามารถ run หรือ execute ได้ หากเป็น directory จะสามารถ cd เข้าไปได้
- 2 = w = Write
- 4 = r = Read
