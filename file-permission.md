

#### ownership

- u = User
- g = Group
- o = Other

#### file type

- <strong>-</strong> = File
- d = Directory
- l = Symbolic link

#### numberic code

- 0 = No Permission
- 1 = x = Execute
- 2 = w = Write
- 4 = r = Read
